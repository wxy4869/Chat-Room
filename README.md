# Chat-Room

> 二春 JAVA 期末作业



## 功能实现

1. 注册
2. 登录
3. 退出
4. 查看在线用户
5. 发起私聊
6. 服务器广播通知
7. 服务器群聊用户
8. 服务器私聊用户
9. 查看当前链接客户端信息
10. 传输文件
11. 发送语音



## 技术

- Java IO
- 多线程
- 网络编程 Socket



## 环境

- Java 8
- gradle 7.1.1
